# Black Holes Inside and Out II website

Static conference website for GitHub Pages, Netlify, or Cloudflare Pages.

## Quick edit
Edit the HTML files directly in GitHub or with any text editor.

## Publish with GitHub Pages
1. Create a new GitHub repository, e.g. `bhio27`.
2. Upload all files in this folder to the repository root.
3. In GitHub: Settings → Pages.
4. Under “Build and deployment”, choose “Deploy from a branch”.
5. Choose branch `main` and folder `/root`.
6. Save. The site will appear at `https://YOUR-USERNAME.github.io/bhio27/`.

## Main files
- `index.html`: home page
- `about.html`: about, committees, code of conduct
- `program.html`: tentative program
- `speakers.html`: invited speakers
- `participants.html`: participant list
- `posters.html`: poster information
- `registration.html`: registration information
- `accommodation.html`: venue and travel info
- `explore-rome.html`: Rome recommendations
- `assets/style.css`: visual design
- `assets/script.js`: mobile menu
